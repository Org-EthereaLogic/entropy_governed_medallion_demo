# Security Policy

## Supported Version

Security fixes are applied to the latest state of the `main` branch.

## Reporting a Vulnerability

Do not open a public GitHub issue for sensitive disclosures.

Report suspected vulnerabilities by emailing `anthony@etherealogic.com` with the subject line `SECURITY: entropy_governed_medallion_demo`.

Please include:

- A concise description of the issue
- Reproduction steps or a proof of concept
- Impact assessment
- Any suggested remediation

Avoid sending client data, secrets, or other sensitive third-party material.

## Response Goals

The project will aim to acknowledge reports within 5 business days and provide status updates as triage progresses.
