# Support

## Getting Help

- **Bug reports**: Open an issue using the [Bug Report template](https://github.com/Org-EthereaLogic/entropy_governed_medallion_demo/issues/new?template=bug_report.md).
- **Feature requests**: Open an issue using the [Feature Request template](https://github.com/Org-EthereaLogic/entropy_governed_medallion_demo/issues/new?template=feature_request.md).
- **Questions**: Open a [Discussion](https://github.com/Org-EthereaLogic/entropy_governed_medallion_demo/discussions), start with the [README](README.md) for the executive overview, and use [docs/technical-approach.md](docs/technical-approach.md) for implementation detail.
- **Security vulnerabilities**: See [SECURITY.md](SECURITY.md) for private reporting instructions.

## Before Opening an Issue

1. Search existing [issues](https://github.com/Org-EthereaLogic/entropy_governed_medallion_demo/issues) to avoid duplicates.
2. Run `make test` and `make lint` to confirm the issue reproduces on a clean install.
3. Include your Python version, OS, and the full error traceback.
